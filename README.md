# Transient Heat Conduction Through Composite Walls

Interactive simulation of transient heat conduction through a multi-layer composite wall using the Finite Difference Method (FDM).

## Features
- Multi-layer wall
- Different materials
- Transient conduction
- Convective cooling
- Streamlit GUI
- Real-time visualization

## Run Locally

```bash
streamlit run advanced_transient_composite_wall.py
